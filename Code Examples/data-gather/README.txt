DataGather

This is a website designed to showcase keyboard controls and use of cookies with Javascript.

- Info button next to <study name> provides controls
- Buttons to remove cell highlights and cell locks appear and disappear based on how many cells are affected.
- User selections are stored as cookies and will reload when the page also reloads.

NOTE: Hyperlinks do not go anywhere.
